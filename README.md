# ライフアップ専用アプリ（LifeUp App）

株式会社ライフアップ 従業員向け 業務モチベーション支援アプリケーション。

従業員一人ひとりが個別アカウントを保有し、AIキャラクターとのチャットやキャラクター育成（レベル上げ）を通じて、日々のモチベーション向上と自己成長を促進します。

## 主な機能

| 機能 | 説明 |
| --- | --- |
| スプラッシュ画面 | 御社キャラクター画像を中央に表示するブランドローディング画面 |
| ログイン / 認証 | Supabase 認証（アカウント作成はリーダー以上の権限者） |
| ダッシュボード | 経験値推移グラフ・レベル・ランキングの可視化 |
| AIチャット | キャラクターと LINE 風 UI で対話（テキスト・音声） |
| キャラクター | 名前・性格・髪色・衣装のカスタマイズとステータス表示 |
| 経験値・レベル・ランク | 業務成果と日々の行動による経験値・月次ランク |
| ラッピーコイン | 社内通貨の残高・推移・取引履歴 |
| ランキング | 全社の月間 / 累計ランキング（表彰台演出） |
| 日報・AIアドバイス | よかった / 悪かったこと・原因の記録と AI からの助言 |
| シフト管理 | カレンダー表示（スプレッドシート連携想定） |
| 設定 | プロフィール・APIキー・暗証番号・通知設定 |

## 技術スタック

- **Next.js 14**（App Router）+ TypeScript
- **Tailwind CSS** + **Framer Motion**（モーション / アニメーション）
- **Supabase**（認証・データベース）
- **Zustand**（状態管理）
- **Recharts**（グラフ可視化）
- **Lucide React**（アイコン）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 環境変数の設定（.env.example をコピー）
cp .env.example .env.local
# .env.local に Supabase の URL と anon key を記入

# 開発サーバー起動
npm run dev
```

ブラウザで http://localhost:3000 を開きます。

## キャラクター画像の追加

スプラッシュ画面に表示する御社キャラクター画像は `public/character.png` に配置してください。
詳細は [`public/README.md`](./public/README.md) を参照。

## ビルド

```bash
npm run build
npm start
```
