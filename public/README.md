# 画像配置ガイド

## キャラクター画像（スプラッシュ画面）

ロード画面（スプラッシュ）に表示される御社キャラクターの画像は、このフォルダに配置してください。

### 配置手順

1. キャラクター画像を以下のファイル名で、このフォルダ（`public/`）に追加してください:

   ```
   public/character.png
   ```

2. 推奨仕様:
   - 形式: PNG（透過背景推奨）または JPG
   - サイズ: 512 × 512 px 以上の正方形
   - ファイル名: 必ず `character.png`

3. 画像を追加してコミット・プッシュすると、スプラッシュ画面（`/`）の中央に自動で表示されます。

> 画像が無い場合は、フォールバックとして 🌟 アイコンが表示されます。
> 画像を配置すると自動的にそちらが優先表示されます。

### 表示箇所のコード

スプラッシュ画面: `src/app/page.tsx` の `<Image src="/character.png" ... />`
